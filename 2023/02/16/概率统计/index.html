概率统计

一 随机事件的概率

1.1 随机事件与样本空间

区分互斥和对立：

  • 互斥：$A \cap B = \phi$
  • 对立：$A \cap B = \phi 且 A \cup B = S$

运算律：

$A - B = A\overline{B} = A - (AB)$

$\overline{U_{i=1}^n A_i} = \cap_{i = 1}^n \overline{A_i}$

$\overline{\cap_{i=1}^n A_i} = \cup_{i = 1}^n \overline{A_i}$

1.2 概率的定义及其性质

  • 古典概型：事件个数有限，每个事件等可能发生
  • 古典概型，几何概型：要求等可能
  • 频率：$f_n(A) = \frac{n_a}{n}$
  • 概率：频率围绕概率波动，频率趋于概率

1.3 概率的功利化定义及概率的性质

  • $P(\overline{A}) = 1 - P(A)$
  • 若$A \subset B$，$P(B - A) = P(B) - P(A)$
  • $P(\overline{A}B) = P(B - A) = P(B - AB) = P(B) - P(AB)$
  • $P(A \cup B) = P(A) + P(B) - P(AB)$
  • $A_{1} \subset A_{2} \subset A_{3} \subset \cdots \subset A_{n} \subset \cdots,B = \Sigma_{i = 1}^{+\infty}A_{i},则\mathop{lim}\limits_{n \rightarrow +\infty}P(A_{n}) = P(B)$
  • $A_{1} \supset A_{2} \supset A_{3} \supset \cdots \supset A_{n} \supset \cdots,B = \prod_{i = 1}^{+\infty}A_{i},则\mathop{lim}\limits_{n \rightarrow +\infty}P(A_{n}) = P(B)$
  • $P(\Sigma_{i = 1}^{+\infty}A_{i}) ≤ \Sigma_{i = 1}^{+\infty} P(A_{i}) $
  • 注意：n个人绕成圈，共有$\frac{n!}{n} = (n-1)!$种排法

1.4条件概率与概率的乘法公式

  • $P(A|B) = \frac{P(AB)}{P(B)}$
  • $P(A_1 \cup A_2 | B) = P(A_1|B) + P(A_2|B) - P(A_1A_2|B) $
  • $P(\overline A | B) = 1 - P(A|B)$
  • $P((A_1 - A_2) | B) = P(A_1|B) - P(A_1A_2|B) $
  • $P(A_1|B)= P((A_1 - A_2) | B) + P(A_1A_2|B) $
  • $P(AB) = P(B)P(A|B)$
  • $P(AB) = P(A)P(B|A)$
  • $如果P(A_1A_2\cdots A_{n-1}) > 0,则P(A_1A_2\cdots A_n) = P(A_1)\cdot P(A_2|A_1)\cdot P(A_3|A_1A_2)\cdots P(A_n|A_1A_2\cdots A_{n-1})$
  • $如果P(BA_1A_2\cdots A_{n-1}) > 0,则P(A_1A_2\cdots A_n|B) = P(A_1|B)\cdot P(A_2|BA_1)\cdot P(A_3|BA_1A_2)\cdots P(A_n|BA_1A_2\cdots A_{n-1})$

1.5全概率公式与贝叶斯公式

  • $\Sigma_{i=1}^nB_i = S;B_1,B_2,\cdots B_n互不相容;P(B_i) > 0;则,P(A) = \Sigma_{i = 1} ^ nP(B_i)P(A|B_i)$
  • $\Sigma_{i=1}^nB_i = S;B_1,B_2,\cdots B_n互不相容;P(B_i)>0;P(A)>0;则,P(B_i|A) = \frac{P(B_i)P(A|B_i)}{\Sigma_{i = 1} ^ nP(B_i)P(A|B_i)}$

1.6事件的独立性

  • $A,B独立,则P(AB) = P(A)P(B)$

二 随机变量及其分布

2.1随机变量

2.2随机变量的分布函数

  • $F(X)右连续$
  • $\lim\limits_{x\rightarrow+\infty}F(x) = 1,\lim\limits_{x\rightarrow-\infty}F(x) = 0$

2.3离散型随机变量及其概率分布

2.4常用离散型随机变量的分布律
0-1分布 $P{x = k} = p^k(1-p)^{1-k}$
二项分布 $P{x = k} = C_n^kp^k(1-p)^{1-k}$ $$B(N,P)\最大值:(n+1)p-1 \le k \le (n+1)p$$
超几何分布 $P{x=k}=\frac{C_N^kC_M^{n-k}}{C_{M+N}^{n}}$ M件正品，N件次品，从中取n件，取到次品数
泊松分布 $P{x = k} = e ^{-k}\frac{\lambda^k}{k!}$ $\Pi(\lambda)$

超几何分布的极限是二项分布，二项分布的极限是泊松分布

2.5连续型随机变量及其概率密度函数

  • $F(x) = \int_{-\infty}^{x}f(x)dx$
  • $\int_{-\infty}^{+\infty}f(x)dx = F(+\infty) = 1$
  • $P(x = a) = 0$，概率为0的事件不一定不发生，概率为1的事件不一定发生

2.6常用的连续型随机变量分布
均匀分布 $$f(x) = \begin{cases} \frac{1}{b-a} , a < x < b \ 0 , 其它\end{cases}$$ $$F(x) = \begin{cases} 0,x<a\ \frac{x - 1}{b - a}, a \le x \le b \ 1, x > b\end{cases}$$ $X \sim U(a,b)$
指数分布 $$f(x) = \begin{cases} \lambda e^{-\lambda x},x > 0\ 0,其它\end{cases}$$ $$F(x) = \begin{cases} 0, x \lt 0\ 1-e^{-\lambda x},x \ge 0 \end{cases}$$ $X \sim E(\lambda) \ P(X > s + t

2.7正态分布

  • $f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma ^2}} $
  • $\int_{-\infty}^{+\infty}e^{-x^2}dx = \sqrt{\pi}$
  • $拐点:x = \mu \pm \sigma$
  • $F(\mu) = \frac{1}{2}$
  • $标准正太:f(x) = \frac{1}{\sqrt{2\pi}}e^{-\frac{x^2}{2}}$
  • $\Phi(x) = 1 - \Phi(-x)$
  • $X\sim N(\mu,\sigma^2),F(x) = \Phi(\frac{x-\mu}{\sigma})$
  • $z_\alpha = - z_{1-\alpha}$
  • $P{|X| > z_{1 - \frac{\alpha}{2}}} = \alpha$

三 二维随机变量

3.1 随机向量与联合分布

离散

  • $固定y,F(-\infty,y) = 0$
  • $固定x,F(x,-\infty) = 0$
  • $F(-\infty,-\infty) = 0$
  • $F(+\infty,+\infty) = 1$
  • $P {x_1 \lt x \lt x_2,y_1 \lt y \lt y_2} = F(x_2,y_2) - F(x_2,y_1) - F(x_1,y_2) + F(x_1,y_1)$
  • $F(x,y) = \mathop{\sum}\limits_{x_i<= x}\mathop{\sum}\limits_{y_i <= y}p_{ij} $

连续

  • $F(x,y) = \int_{-\infty}^{x}\int_{-\infty}^{y} f(u,v) dudv$
  • $P(X = a,Y = b) = 0 = P((X,Y) = (a,b))$
  • $P(X = a,-\infty \lt Y +\infty) = 0$
  • $P(-\infty \lt X +\infty,Y = b) = 0$
二维均匀分布 $f(x,y) = \begin{cases} \frac{1}{A},(x,y)\in D\0,其他 \end{cases}$ $A为D的面积,\(X,Y) \sim U(D), \ 任意D_1 \in D,D_1面积A_1,P((x,y)\in G_1) = \frac{A_1}{A} $
二维正态分布 $f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}}e^{-\frac{1}{2(1-\rho^2)}[\frac{(x - \mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}]}$ $(X,Y) \sim N(\mu_1,\sigma_1^2;\mu_2,\sigma_2^2;\rho),-1<\rho<1$

3.2边缘分布函数

  • $F_x(x)=\mathop{lim}\limits_{x\rightarrow +\infty}F(x,y) = F(x,+\infty)$
  • $F_y(y)=\mathop{lim}\limits_{y\rightarrow +\infty}F(x,y) = F(+\infty,y)$

3.3 边缘分布率与条件分布律

离散

  • $P{X=x_i} = \mathop{\sum} \limits_jP{X=x_i,Y = y_j} = \mathop{\sum}\limits_{j}p_{ij}$
  • $P{Y=y_j} = \mathop{\sum} \limits_iP{X=x_i,Y = y_j} = \mathop{\sum}\limits_{i}p_{ij}$

3.4 边缘概率密度与条件概率密度

  • $F_X(x) = \int_{-\infty}^{x}\int_{-\infty}^{+\infty}f(u,v)dvdu$

  • $f_X(x) = \int_{-\infty}^{+\infty}f(x,y)dy$

  • $F_X(x) = \int_{-\infty}^{x}\int_{-\infty}^{+\infty}f(u,v)dvdu$

  • $f_Y(y) = \int_{-\infty}^{+\infty}f(x,y)dx$

  • $(X,Y) \sim N(\mu_1,\sigma_1^2;\mu_2,\sigma_2^2;\rho),则X \sim (\mu_1,\sigma_1^2),Y \sim (\mu_2,\sigma_2^2)$

边平行于坐标轴的矩形域上的均匀分布的边缘分布仍为均匀分布，二维正态分布的边缘分布仍为正态分布

离散条件概率

  • $P{Y = y_j |X= x_i} = \frac{P{X = x_i,Y= y_j}}{P{X = x_i}} = \frac{p_{ij}}{\mathop\sum\limits_{j}^{+\infty}p_{ij}}$
  • $P{X = x_i |Y= y_j} = \frac{P{X = x_i,Y= y_j}}{P{Y = y_j}} = \frac{p_{ij}}{\mathop\sum\limits_{i}^{+\infty}p_{ij}}$
  • $F_{X|Y}(x|y) = \frac{\frac{\partial}{\partial y}F(x,y)}{\frac{d}{dy}F_Y(y)} = \frac{\int_{-\infty}^x f(u,y)du}{f_Y(y)}$
  • $F_{Y|X}(y|x) = \frac{\frac{\partial}{\partial y}F(x,y)}{\frac{d}{dy}F_X(x)} = \frac{\int_{-\infty}^y f(x,v)dv}{f_X(x)}$
  • $f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)}$
  • $f_{Y|X}(y|x) = \frac{f(x,y)}{f_X(x)}$

3.5 相互独立的随机变量

  • $独立,F(x,y) = F_X(x)F_Y(y),P{a \lt x \le b,c \lt y \le d} = P{a \lt x \le b}P{c \lt y \le d}$
  • $离散,P{X = x_i,Y = y_j} = P{X = x_i}P{Y = y_j} $

五 随机变量的数字特征

5.1 随机变量的数学期望

不是所有随机变量都有数学期望：Cauchy分布期望不存在

1. 离散型

$$
E(X) = \mathop{\Sigma}\limits_{k=1}^{+∞}x_kp_k
$$

2. 连续型

$$
E(X) =∫_{-∞}^{+∞}xf(x)dx
$$

3.随机变量函数

  • 一维离散型

​ $P(X=x_i) = p_i(i = 1,2,\cdots),Y = g(X)$
$$
E(Y) =\Sigma_{i=1}^{+∞}g(x_i)p_i
$$

  • 一维连续型

    $Y = g(X)$

$$
E(Y) = ∫_{-∞}^{+∞}g(x)f(x)dx
$$

  • 二维离散型

​ $P(X=x_i,Y = y_j) = p_{ij}(i,j = 1,2,\cdots),Z = g(X,Y)$
$$
E(Z) =\Sigma_{i=1}^{+∞}\Sigma_{j=1}^{+∞}g(x_i,y_j)p_{ij}
$$

  • 二维连续型

    $Z = g(X,Y)$

$$
E(Z) = \int_{-\infty}^{\infty}\int_{-\infty}^{\infty}g(x,y)f(x,y)dxdy
$$

4.数学期望的性质

  1. $E(C) = C$

  2. $E(aX) = a E(X)$

  3. $E(X + Y) = E(x) + E(Y)$

  4. $E(\Sigma_{i = 1}^na_iX_i+C) = \Sigma_{i = 1}^{n}a_iE(Xi) + C$

  5. ==X,Y独立时$E(XY) = E(X) E(Y)$==

    ==注意：E(X) E(Y) = E(XY)不能推出XY独立==

  6. 若存在常数a使P(X &ge; a) = 1,则E(X) &ge; a

    若存在常数a使P(X &le; b) = 1,则E(X) &le; b

5.例题

将 4 个可区分的球随机地放入 4 个盒子中，每盒容纳的球数无限，求空着的盒子数的数学期望.
$$
引入x_i，i=1,2,3,4 \
xi = \begin{cases} 1,&第i盒空\0,&第i盒非空\end{cases}\
X = x_1 + x_2 + x_3 + x_4\
P= \begin{cases} (\frac{3}{4})^4,&x=1\1 - (\frac{3}{4})^4,&x = 0\end{cases}\
E(x_i) = (\frac{3}{4})^4 \
E(X) = 4\times(\frac{3}{4})^4 = \frac{81}{64}
$$

5.2 方差

$ D(X) = E((X - E(X))^2)$

1.离散型

$$
D(X) = \Sigma_{k = 1}^{+\infty}(x_k - E(X))^2p_k
$$

2.连续型

$$
D(X) = \int_{-\infty}^{+\infty} (x_k - E(X))^2f(x)dx
$$

3. 常用计算公式

$$
D(X) = E(X^2) - (E(X))^2
$$

4. 方差的性质

  1. $D(C) = 0$

  2. $D(aX) = a^2 D(X)$

  3. $D(aX + b) = a^2D(X)$

  4. $D(X ± Y) = D(X) + D(Y) ± 2E((X - E(X))(Y - E(Y)))\=D(X) + D(Y) ± 2Cov(X,Y) \= D(X) + D(Y) ± (E(XY) - E(X)(E(Y))$

  5. ==XY独立$D(X ± Y) = D(X) + D(Y)$==

    D(X ± Y) = D(X) + D(Y)不能推出X,Y独立

  6. 任意常数C，$D(X)$ &le; $E(X-C)^2$ 当且仅当 $C = E(x)$时，等号成立

  7. D(X) = 0 当且仅当 P{X = E(x)} = 1

7.3 常用统计量的分布

  • 正态总体样本的线性函数的分布

    • $\overline X \sim (\mu,\frac{\sigma^2}{n})$

  • $\chi^2分布$

    • $X_1,X_2, \cdots X_n相互独立，且都 \sim N(0,1),则\chi^2 = \mathop\sum\limits_{i=1}^nX_{i}^2,称\chi^2服从自由度n为2的\chi^2分布,记作\chi^2 \sim \chi^2(n)$
    • $E(\chi^2(n)) = n,D(\chi^2(n)) = 2n$.
    • $若X_1 = \chi^2(n_1),X_2 = \chi^2(n_2),且X_1,X_2独立，则X_1+X_2 \sim \chi^2(n_1+n_2)$.
    • $n \rightarrow \infty, \chi^2(n) \rightarrow 正态分布$.
    • $X_1,X_2, \cdots X_n相互独立，且都 \sim N(\mu,\sigma^2),则\ 1.\overline X与 S^2 相互独立 \2.\space\frac{(n-1)}{\sigma^2}S^2 \sim \chi^2(n-1)$
    • $X_1,X_2, \cdots X_n相互独立，且都 \sim N(\mu_i,\sigma_i^2),则\ \mathop\sum\limits_{i=1}^n(\frac{X_i -\mu_i}{\sigma_i})^2 \sim \chi^2(n)$

  • t分布

    • $X \sim N(0,1),Y \sim \chi^2(n),X,Y相互独立， T = \frac{X}{\sqrt{\frac{Y}{n}}},T服从的分布成为自由度为n的t分布,记为T \sim t(n),f(t)是偶函数$
    • $t_{1-\alpha}(n) = -t_\alpha(n)$
    • $P{|T| \ge t_{1-\frac{\alpha}{2}}(n) } = \alpha$
    • $X_1,X_2, \cdots X_n相互独立，且都 \sim N(\mu,\sigma^2),则\ \frac{\overline X - \mu}{\frac{S}{\sqrt{n}}} \sim t(n-1)$
    • $X_1,X_2, \cdots X_n 和 Y_1,Y_2, \cdots Y_n 分别是服从正态主体 N(\mu_1,\sigma^2)和N(\mu_2,\sigma^2)中抽取的独立样本，则 \T = \frac{(\overline X - \overline Y) - (\mu_1 - \mu_2)}{\sqrt{(m-1)S_1^2} + (n-1)S_2^n} \cdot \sqrt{\frac{mn(m+n-2)}{m+n}} \sim t(m+n-2) $

  • F分布

    • $设 X \sim \chi^2(n),Y \sim \chi^2(m),X,Y相互独立,F = \frac{\frac{X}{n}}{\frac{Y}{m}},则F服从第一自由度为n,第二自由度为m的F分布,记为F \sim F(n,m)$
